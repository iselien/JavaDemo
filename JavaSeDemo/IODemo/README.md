# IODemo

### 1.1 概述与分类

Java中的IO流技术主要用于传输数据。典型的应用场景有：读写本地文件、上传下载文件等。按照数据传输的方向可以分为两种：

- **输入流(Input)**：即让数据进入应用程序中。
- **输出流(Output)**：即让数据离开应用程程序。

按照数据的种类还可以有另一外一种分法：

- **字节流**：可以处理任何一种数据。
- **字符流**：只能处理纯文本数据。
